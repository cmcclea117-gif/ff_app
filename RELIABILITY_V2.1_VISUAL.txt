## Button State Examples

### When "Overall" Mode Active
```
┌─────────────────────────────────────────────────────────┐
│ 🏆 Top 20 Most Reliable Players                         │
│                                                          │
│  [🌍 Overall] ← Active (Green border, bold)            │
│  [💎 Available] ← Inactive (Blue border, lighter)      │
└─────────────────────────────────────────────────────────┘

Top 20 shows: All players (⭐ = rostered)
```

### When "Available" Mode Active
```
┌─────────────────────────────────────────────────────────┐
│ 🏆 Top 20 Most Reliable Players                         │
│                                                          │
│  [🌍 Overall] ← Inactive (Green border, lighter)       │
│  [💎 Available] ← Active (Blue border, bold)           │
└─────────────────────────────────────────────────────────┘

Top 20 shows: Only available players (no stars needed)
```

## Filter Update Logic Visual

```
┌─────────────────────────────────────────────────────────────────┐
│ FILTER CHANGE FLOWCHART                                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Change Position? ────────→ ✅ Updates Top 20                   │
│                     └─────→ ✅ Updates Table                    │
│                                                                  │
│  Change Min Games? ───────→ ✅ Updates Top 20                   │
│                     └─────→ ✅ Updates Table                    │
│                                                                  │
│  Change Min Points? ──────→ ✅ Updates Top 20                   │
│                     └─────→ ✅ Updates Table                    │
│                                                                  │
│  Toggle Overall/Available? → ✅ Updates Top 20                   │
│                     └─────→ ❌ Does NOT update Table            │
│                                                                  │
│  Type in Search? ─────────→ ❌ Does NOT update Top 20           │
│                     └─────→ ✅ Updates Table                    │
│                                                                  │
│  Change Roster Status? ───→ ❌ Does NOT update Top 20           │
│                     └─────→ ✅ Updates Table                    │
│                                                                  │
│  Click Reset? ────────────→ ✅ Updates Top 20 (to Overall)      │
│                     └─────→ ✅ Updates Table (all filters)      │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## Example Scenarios

### Scenario 1: Finding Top Available WRs

**User Actions:**
```
1. Click [💎 Available]
2. Set Position: WR
3. Set Min Games: 5+
4. Set Min Avg Points: 8+
```

**What Updates:**
```
TOP 20:
  ✅ Shows only available WRs
  ✅ With 5+ games played
  ✅ With 8+ PPG
  ✅ Sorted by reliability score
  ✅ No roster stars (all available)

TABLE:
  ✅ Shows only available WRs
  ✅ With 5+ games
  ✅ With 8+ PPG
  ✅ ALL matching players (not just top 20)
```

**Visual Result:**
```
🏆 Top 20 = Your waiver wire priority list (WRs, 5+ games, 8+ PPG)
📊 Table = Full list of all matching players for detailed comparison
```

## Mobile/Responsive Behavior

### Desktop (1800px+)
```
Filters: Single row, all 6 items side-by-side
Top 20: 5 cards per row
```

### Tablet (768px-1800px)
```
Filters: Wraps to 2 rows (3 items per row)
Top 20: 3-4 cards per row
```

### Mobile (<768px)
```
Filters: Stacks vertically (1 per row)
Top 20: 1-2 cards per row
```

## 🎯 Key Takeaways

1. **Toggle buttons** at top-right switch between Overall and Available modes
2. **Horizontal filters** save 110px of vertical space (~61% reduction)
3. **Smart updates**: Position/Games/Points update both Top 20 and table
4. **Isolated updates**: Search/Roster only update table (not Top 20)
5. **Visual feedback**: Active toggle button is brighter and bold
