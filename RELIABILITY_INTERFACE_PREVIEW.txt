## 🎨 Interactive Elements

### Top 20 Grid
- **Responsive:** Adjusts to screen width (4-5 cards per row on desktop)
- **Hover Animation:** Cards scale up 5% on hover
- **Color Coding:** Green/yellow/orange borders match reliability tiers
- **Roster Stars:** ⭐ indicates players on your roster

### Filter Dropdowns
- **Position:** Click to see all 5 options (All, QB, RB, WR, TE)
- **Min Games:** 3 preset options (3+, 5+, 7+)
- **Min Avg Points:** 6 options from "All" to "15+ PPG"
- **Roster Status:** 3 options (All, Rostered, Available)

### Search Box
- **Real-time:** Updates table as you type
- **Case-insensitive:** "LADD" or "ladd" both work
- **Partial matching:** "mc" finds "McConkey" and "McLaurin"

### Table Headers
- **Clickable:** Every column header is sortable
- **Sort Indicator:** Arrow (▼/▲) shows current sort
- **Re-click:** Toggle between ascending/descending

---

## 📱 Responsive Design

### Desktop (1800px+)
```
Top 20 Grid: 5 cards per row
Filter Layout: 5 columns (all filters visible)
Table: Full width with all 9 columns
```

### Tablet (768px-1800px)
```
Top 20 Grid: 3-4 cards per row
Filter Layout: 3 columns, wraps to second row
Table: Scrollable horizontally
```

### Mobile (< 768px)
```
Top 20 Grid: 1-2 cards per row
Filter Layout: 1 column, stacks vertically
Table: Scrollable horizontally with pinned first column
```

## 💡 Color Legend

### Reliability Score Colors
- **🟢 Elite (90-100):** Green border, green icon
- **🟡 Good (75-89):** Yellow border, yellow icon
- **🟠 Fair (60-74):** Orange border, orange icon
- **🔴 Poor (<60):** No border color, red icon

### Position Badge Colors
- **QB:** Blue background (#3498db)
- **RB:** Red background (#e74c3c)
- **WR:** Purple background (#9b59b6)
- **TE:** Teal background (#16a085)

### Bias/Trend Colors
- **📉 Positive (red text):** Player misses projections
- **➡️ Near zero (gray text):** Unbiased
- **📈 Negative (green text):** Player beats projections

### Roster Indicators
- **⭐ Star:** Player is on your roster (visible in both Top 20 cards and table)

## 🎬 The Complete Experience

When you open the Reliability tab, you see:

1. **Top 20 Grid** (always visible, unfiltered)
   - Shows the 20 most reliable players league-wide
   - Your rostered players marked with ⭐
   - Color-coded by reliability tier
   - Hover for visual feedback

2. **Explainer Section**
   - Quick reference for score interpretation
   - What each tier means

3. **Powerful Filters**
   - 5 different filter types
   - Combine for surgical precision
   - Real-time updates
   - One-click reset

4. **Sortable Data Table**
   - Click any column to sort
   - Visual sort indicators
   - Applies to filtered data
   - Shows count of filtered results

**Result:** You have complete control to find exactly the players and insights you need! 🎯

3. **Combine roster + position:** See only YOUR QBs or YOUR RBs
4. **Search is instant:** No need to press enter, just start typing
5. **Reset button is your friend:** When table shows no results, click reset and try again
